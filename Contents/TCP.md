# TCP 통신
TCP란 소켓 프로그래밍 중의 하나로 Stream 통신 프로토콜이라고 부르며, 양쪽의 소켓이 연결된 상태여야만 통신이 가능하기 때문에 연결지향 프로토콜이라고 한다.<br>
TCP 프로토콜은 신뢰성 있는 프로토콜이기 때문에, 송신한 쪽의 데이터가 수신 측에 차례대로, 중간에 유실되는 일 없이 도착하는 것을 의미한다.
# 게이트웨이 (Gateway)
게이트웨이를 다른 말로 라우터라고 한다. 소프트웨어 측을 강조할 때 게이트웨이라고 하고, 하드웨어적 측면을 강조할 때 라우터라고 한다.<br>
게이트웨이 혹은 라우터는 인터넷으로 접속하기 위한 일종의 관문의 역할을 한다고 이해하면 된다.<br>
라우터가 인터넷 공간에서 각기 다른 호스트 사이를 연결해주는 기능을 라우팅(routing)이라고 한다.
